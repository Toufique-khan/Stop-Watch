# ⏱️ Stopwatch Web App

A fully responsive and interactive **Stopwatch Web Application** built using **HTML, CSS, and JavaScript**.

## 🚀 Features
- ✅ Start / Stop Functionality
- ✅ Reset Timer
- ✅ Accurate Time Calculation
- ✅ Light & Dark Mode Toggle 🌙☀️
- ✅ Responsive Design for All Devices
- ✅ Clean and Modern UI

## 🧠 What I Learned
Building this project helped me improve my understanding of:
- JavaScript DOM Manipulation
- Event Handling
- Timer Functions (`setInterval`)
- Time Calculations & Formatting
- UI State Management
- Theme Switching Logic
- Responsive Web Design

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
